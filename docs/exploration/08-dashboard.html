08 — Dashboard Page
WorldCup 2026 · Architecture Explorer
React App Database Supabase
Steps
Layers
React App
Dashboard.jsx is the main page. Renders hero countdown, global leaderboard (always), today's games strip, and My Stats panel. All state managed locally with useState + useEffect.
Database
RPCs: get_leaderboard() for global standings, delete_account() for self-service account removal. Games fetched from games table with date filter.
Supabase
Auth session read via supabase.auth.getSession(). Table queries via Supabase JS client. Username update goes to both auth.users (metadata) and profiles table.
