Steps
Layers
Landing — Vanilla JS
Plain HTML + CSS files served statically. app.html is the SPA entry point built by Vite. css/style.css is shared between landing and React app.
React App
Vite SPA at app.html. React 18 + HashRouter. Context providers wrap the whole app. All inner pages live here.
Supabase
Cloud backend: Auth, DB, Realtime. ESM npm client used in the React app (different from the UMD CDN client in the landing page).
Database (PostgreSQL)
Supabase-hosted Postgres. RLS policies guard all tables. Accessed via SDK or stored RPCs.
