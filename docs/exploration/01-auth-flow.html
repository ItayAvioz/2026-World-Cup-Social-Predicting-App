Landing (Vanilla) React App Supabase Database
Steps
Layers
Landing — Vanilla JS
Plain index.html + js/auth.js. No framework. Runs in the browser directly. Handles register, login, and invite before the React app is involved.
React App
Vite SPA at app.html. React 18 + HashRouter. All inner pages (Dashboard, Groups, Picks…). Manages session via AuthContext and guards routes via AuthGuard.
Supabase Auth
Cloud auth service. Manages auth.users, issues JWT tokens, stores session in localStorage automatically. Never exposes raw passwords.
Database (PostgreSQL)
Supabase-hosted Postgres. Public tables (profiles, groups, predictions…). RLS policies enforce who can read/write each row. Accessed via SDK or RPCs.
