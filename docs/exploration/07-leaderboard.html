07 — Leaderboard (Global + Group)
WorldCup 2026 · Architecture Explorer
Database React App Supabase
Steps
Layers
Database (PostgreSQL)
Two RPCs: get_leaderboard() (global) and get_group_leaderboard(group_id). Both use RANK() OVER with exact_scores tiebreaker. RLS ensures only group members see group data.
React App
Dashboard.jsx shows global leaderboard always (no toggle). Groups.jsx shows per-group leaderboard inside each group card. Both highlight the current user's row(s).
Supabase
RPC calls from React. RLS on group_members enforces group privacy. Polling (60s) keeps leaderboard fresh during active game days.
