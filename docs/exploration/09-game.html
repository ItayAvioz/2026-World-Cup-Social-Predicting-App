React App
Game.jsx is a single page for one game. Reads game ID from URL params, resolves group context, fetches team stats, odds, and predictions. Handles predict widget, reveal, stat blocks, and knockout display.
Supabase / RLS
RLS on predictions enforces the kickoff reveal: kick_off_time <= now() must be true AND the viewer must share a group with the prediction owner. Server-side NOW() used — no client clock trust.
Database
Tables: games, team_tournament_stats, predictions. Prediction upsert uses UNIQUE(user_id, game_id, group_id). Knockout fields: went_to_extra_time, et_score_home/away, went_to_penalties, penalty_score_home/away.
